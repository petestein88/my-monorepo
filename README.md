# My Monorepo

This monorepo contains several projects organized into subdirectories:

## Subdirectories

- **iot**: Contains the IoT backend services and related code.
- **iot-frontend**: Contains the frontend application for interacting with the IoT services.

## Structure

```
my-monorepo/
├── iot/
│   └── ...
└── iot-frontend/
    └── ...
```

Use this monorepo to manage both IoT backend and frontend in a cohesive way, facilitating easier updates and collaboration.
